## Myorep Defaults

1. `barbell_curl`
2. `lateral_raise`
3. `incline_dumbbell_curl`

## Substitution Notes

1. If bench is unavailable:
   - `incline_dumbbell_press` -> `dumbbell_neutral_floor_press`
2. If elbows are irritated:
   - reduce myorep density before swapping the exercise
3. If triceps overhead work is irritated:
   - `overhead_dumbbell_triceps_extension` -> `barbell_standing_extension`
