## Objective

1. Main goal: hypertrophy plus fat loss.
2. Block length: 12 weeks.
3. Week 13: deload.
4. Next cycle should be adapted from logged performance, fatigue, and recovery notes.

## Operating Constraints

1. Arms are the specialization priority.
2. Sessions should stay at 4 exercises.
3. Bench setup is limited to 2 days per week.
4. Wednesday and Sunday are default rest anchors.
5. Missed sessions roll forward instead of being skipped.
6. Time efficiency matters: pair work when practical and avoid unnecessary setup churn.

## Weekly Structure

1. `M1`: Monday, Bench 1
2. `M2`: Tuesday, DB
3. `M3`: Thursday, BB + DB
4. `M4`: Friday, Bench 2
5. `O5`: Saturday, DB optional

## Session Templates

### M1: Mon (Bench 1)

1. `A1` DB Incline Press: `4 x 10-15`
2. `A2` Hammer Curl myorep: `10-15 + 3-5 x 3-5`
3. `B1` Close-Grip BB Press: `4 x 10-15`
4. `B2` BB Curl / Cheat Curl: `4 x 6-10`

Equipment:
bench, barbell, dumbbells
