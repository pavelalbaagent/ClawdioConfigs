# 2026-03 Coaching Priors

## Hypertrophy And Progression

1. Use the current program as the first anchor, not abstract optimization.
2. Keep exercise selection stable long enough to judge progression from logs instead of swapping too early.
3. Add reps or improve execution before adding more exercises when equipment and setup efficiency matter.
4. For arm specialization, more volume is only useful when elbows, sleep, and session performance still support recovery.

## Fatigue And Deload Signals

1. A single flat session is not enough to force a redesign.
2. Repeated underperformance, persistent soreness, or falling reps across the same movement pattern are stronger signals.
3. Reduce density or trim low-priority sets before cutting the core movement anchors.
4. Use deload decisions from the canonical rules unless new evidence is written back into the repo.

## Elbow And Irritation Management

1. For mild elbow irritation, reduce myorep density or total arm isolation volume before replacing the main session.
2. Keep warmups and first working sets more controlled before deciding a movement is unusable that day.
3. Prefer swaps that preserve the session goal and equipment footprint.
4. If pressing overhead aggravates the elbow, use the canonical substitution path instead of inventing a new exercise.

## Myoreps And Supersets

1. Keep myoreps on stable, lower-skill isolation movements.
2. Heavy compounds should usually stay as straight sets even when time is tight.
3. Density methods are useful when load jumps are coarse, but they still count against recovery.
4. If an arm myorep slot stalls, first check execution quality, rest discipline, and recent volume before adding more work.

## Trend Notes For This Setup

1. Stable exercise pools, clean logging, and recovery-aware volume management are more valuable here than chasing novelty.
2. Home-gym progression benefits from methods that do not depend on tiny load jumps every session.
3. Because the program is arm-specialized, coaching should bias toward protecting elbows and preserving pressing/curl quality.
