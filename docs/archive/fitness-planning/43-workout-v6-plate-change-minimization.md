# Workout v6 (Plate-Change Minimization Pass)

Last updated: 2026-03-05

## Goal of this pass

1. Reduce setup transitions and plate changes within each session.
2. Keep arm-specialization volume high.
3. Keep bench setup at 2 days per week.
4. Preserve lat-width back emphasis and incline-chest emphasis.

## What changed from v5

1. Structured sessions into explicit load buckets (`DB-H`, `DB-M`, `DB-L`, single BB block).
2. Limited each day to 4-5 movements.
3. Added clear per-day equipment usage map.
4. Kept myoreps only on isolation-friendly movements.

## Efficiency rules

1. Max 2-3 load blocks per session.
2. One heavy barbell block per non-bench day when possible.
3. On bench days, keep barbell block compact and avoid extra one-off barbell loads.
4. Pair movements by shared equipment/load bucket whenever possible.
