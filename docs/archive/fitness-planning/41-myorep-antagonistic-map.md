Last updated: 2026-03-05

## What changed

1. Added explicit mapping for where myoreps should be used.
2. Added explicit antagonistic pair map by session block.
3. Kept low-variety exercise pool and high arm volume priority.

## Best myorep matches for your setup

1. Lateral raise
2. Rear-delt fly
3. Kickbacks
4. Hammer curl
5. Incline DB curl (optional if elbows tolerate)

## Avoid myoreps on these

1. Bench press
2. Close-grip bench press
3. Barbell row
4. Deadlift
5. Squat

## Antagonistic pair map

1. Heavy pair first:
   - Press + row
2. Arm pair second:
   - Biceps anchor + triceps anchor
3. Density pair last:
   - Myorep-friendly isolations
