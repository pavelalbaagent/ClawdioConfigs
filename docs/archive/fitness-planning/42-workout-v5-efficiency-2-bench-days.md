# Workout v5 (Efficiency, 2 Bench Days, Arm Specialization)

Last updated: 2026-03-05

## What was changed from v4c

1. Hard cap of 4-5 exercises per session.
2. Bench setup reduced to exactly 2 days (Monday and Friday).
3. Back pattern shifted toward lats/width emphasis (pullover + lat-biased rows).
4. Chest emphasis shifted to incline/upper-chest dominance.
5. Biceps/triceps split set to cover all heads through the week.

## Session structure logic

1. Start with heavy antagonist pair.
2. Then run biceps/triceps antagonist pair.
3. Finish with myorep-friendly isolation or lower-body sprinkle.
