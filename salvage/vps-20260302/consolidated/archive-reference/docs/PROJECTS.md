# PROJECTS.md

## Goal
Build a portfolio of practical AI projects that improves real skills and hiring attractiveness.

## Project Tracks

### 1) Automation Stack Comparison (Make vs Zapier vs n8n)
- **Objective:** Build the same workflow in all three tools and compare capability, speed, flexibility, and cost.
- **Example workflow:** form/email trigger → AI classification/summarization → notify Slack/WhatsApp → log to sheet/database.
- **Deliverables:**
  - Architecture diagram
  - Side-by-side feature comparison
  - 3 short demo videos/GIFs
  - "When to use which tool" decision guide
- **Status:** Idea (ready to scope)

### 2) Web App for Agent Teams
- **Objective:** A web interface where users submit a task and spawn specialized agent roles (planner, researcher, builder, reviewer).
- **Core features (MVP):**
  - Task intake form
  - Role-based agent orchestration
  - Shared task board + outputs
  - Final synthesized report
- **Deliverables:**
  - MVP deployed
  - 2 realistic demo tasks
  - Brief architecture + tradeoffs write-up
- **Status:** Idea (define stack + MVP boundary)

### 3) YouTube Watchlist Auto-Pipeline (#tools -> #projects)
- **Objective:** Automatically ingest relevant videos, summarize daily, and promote high-value ideas into actionable projects.
- **Core features (MVP):**
  - Channel/watchlist ingestion
  - Daily summary to Slack `#news`
  - Promotion recommendations with ROI/effort + first 60-min action
- **Deliverables:**
  - Working ingestion flow
  - Stable daily digest format
  - Promotion rubric and weekly roll-up
- **Status:** Scoped (ready to implement)
- **Workspace:** `projects/youtube-watchlist-pipeline/`
